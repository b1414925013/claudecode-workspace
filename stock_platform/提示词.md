股票收益记录平台 - 开发提示词
项目概述
我想开发一个股票收益记录平台，用于追踪我的投资收益。

核心需求
1. 多账号管理

支持管理 2 个及以上的股票账号（目前是 2 个，未来可扩展）
可单独查看某个账号的收益情况
可汇总查看所有账号的总收益
2. 持仓管理

记录每只股票的持仓数量、买入成本、平均价
实时计算持仓市值和浮动盈亏
支持持仓成本的分批调整（加仓/减仓）
3. 交易记录

记录完整的买卖历史（时间、价格、数量、手续费）
自动计算每笔交易的盈亏
支持分红记录
4. 统计分析

计算收益率（总收益/总成本）
统计胜率（盈利交易数/总交易数）
展示持仓周期分析
支持按时间段筛选（本月/本年/自定义）
5. 杠杆机制（未来扩展）

支持记录融资/杠杆账户
区分现金账户和杠杆账户
分别计算杠杆账户的保证金、爆仓线等风险指标
技术要求
前端：Vue + TypeScript + Tailwind CSS
后端：Python+fastapi
数据库：MySQL (127.0.0.1:3306 账号：root 密码：root 数据库：stock_platform)
API 设计：RESTful
其他功能建议
数据可视化（收益曲线、资产配置饼图）
导出功能（Excel/CSV）
数据备份
